# TechHub - Store Informatique

Site vitrine moderne pour la vente de produits informatiques avec catalogue dynamique, filtres, recherche et panier simulé.

## 🚀 Fonctionnalités

- **Catalogue dynamique** : Affichage des produits depuis une base de données JavaScript
- **Filtrage par catégorie** : iPhone, Android, Ordinateurs
- **Recherche en temps réel** : Recherche textuelle sur les noms et descriptions
- **Fiches produits détaillées** : Modales avec spécifications techniques
- **Panier simulé** : Ajout/retrait de produits, calcul du total, persistance locale
- **Design responsive** : Mobile-first, adapté à tous les écrans
- **Déploiement CI/CD** : Déploiement automatique sur GitHub Pages

## 🛠 Technologies utilisées

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript Vanilla (ES6+)
- GitHub Actions (CI/CD)
- GitHub Pages (Hébergement)
